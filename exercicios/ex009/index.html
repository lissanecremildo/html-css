Ediçao de listas

Listas ordenadas ou ordered list (ol)

Para criar uma lista ordenada usamos a tag <ol> e para inserir os itens da lista (list item) usamos a tag <li>

  1. acordar
  2. meditar
  3. escovar dentes
  4. tomar banho
  5. matabichar
  6. ir ao serviço

Podmos definir o tipo do conteúdo da lista, usando a variavel type, que recebe numeros (1) por padraro, letras maiusculas (A), letras minusculas (a),
numeros romanos maiusculos (I) e numeros romanos minusculos (i);
podemos tambem a posicao que a lista de começar usando a variavel start

  1. acordar
  2. meditar
  3. escovar dentes
  4. tomar banho
  5. matabichar
  6. ir ao serviço

Listas não ordenadas ou unordered list (ul)

Para criar uma lista nao ordenada usamos a tag <ul> que recebe a variel "disk" (bolinha preta pintada) por padrao
, mas tambem podemos definir as variaveis "circle" e "square"
  • arroz
  • oleo
  • cebola
  • batata
  • frango

Lista de definiçoes dl (definiton list)

Para criar uma lista de definições usamos a tag <dl> e para definir o termo de cada definição (definition term) usamos a dt
e para descrever a definição do termo usamos a tag dd (definition description)
