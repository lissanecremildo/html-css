Links para download

Para criar um link para download de arquivos basta indicar o caminho do arquivo que se pretende baixar
conmo valor do atributo href, depois indicar o nome do arquivo como valor do atributo download
e, por fim, indicar o tipo de arquivo como valor do atributo type
